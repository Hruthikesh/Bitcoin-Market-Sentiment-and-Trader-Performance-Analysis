# Bitcoin Market Sentiment vs Trader Performance Analysis

This project analyzes the relationship between Bitcoin market sentiment and trader performance using the Fear & Greed Index and Hyperliquid trading data. The analysis explores how different market conditions affect profitability, win rates, trade sizes, and overall trading behavior.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
